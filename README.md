## Installation
You can install the library via npm/yarn:
```sh
npm install textify.js
```
```sh
yarn add textify.js
```

or via CDN:
```html
<script src="https://cdn.jsdelivr.net/npm/textify.js@1.1.0/dist/Textify.min.js"></script>
```

Import Textify.js:
```javascript
import Textify from "textify.js";
```
