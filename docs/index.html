Text Animations

Textify.js is a simple library that allows you to create text animations, and user can modify the animation for themself.

Scroll-base Animations

Textify.js is a simple library that works with scroll-base animations, and user can use any scrolling libraries likes locomotive-scroll, smoothscroll, etc. All libraries are compatible with Textify.js.

Custom Animations

Textify.js is a provide a simple way to create custom animations. Textify give rotation, scale, translate and fade options to create your own animations, and other new options are add in future.

Custom Eases

Textify.js is a provide 20+ custom eases for create animations and fade animations.
